Privilege Escalation and Persistence

Privilege Escalation

Privilege escalation occurs when a user gains higher permissions than intended. This can happen due to weak permissions or vulnerable services.

Techniques

- Exploiting SUID binaries
- Weak file permissions
- Kernel vulnerabilities

Persistence

Persistence ensures long-term access by:

- Creating cron jobs
- Modifying startup scripts
- Adding malicious services

Living-off-the-Land

Using built-in tools like Bash and PowerShell to avoid detection.
