# Capstone Project: Full VAPT Cycle

## Objective
To simulate a complete penetration testing cycle including scanning, exploitation, and reporting.

---

## Methodology
The testing followed standard phases:
1. Reconnaissance
2. Scanning
3. Exploitation
4. Post-Exploitation
5. Reporting

---

## Findings
- SQL Injection vulnerability detected
- Weak credentials identified
- XSS vulnerabilities present

---

## Impact
Attackers can extract sensitive data, hijack sessions, and gain unauthorized access.

---

## Recommendations
- Implement input validation
- Use strong authentication
- Regularly update systems
- Conduct periodic security testing

---

## Conclusion
The system contains multiple vulnerabilities and requires immediate security improvements.
